Cara Untuk Mengupload di Github

- Buka folder, klik kanan pilih Git Bush Here
- Ketikkan Perindah sbg berikut

git init
git add .
git commit -m "Masukkan Perintah yang kamu Inginkan"
git remote add origin <link github yang sudah dibuat>
git push -u origin master

Maka sudah terupload
